## 📚 Notas Teóricas
### ¿Cómo funciona JavaScript?
* JavaScript es un lenguaje de **alto nivel**, lo que significa que es fácil de leer para los humanos.
* Se ejecuta a través de motores como **V8** (en Chrome y Node.js).
* Diferencia entre **Navegador** (interacción con el usuario) y **Node.js** (servidor/archivos).
